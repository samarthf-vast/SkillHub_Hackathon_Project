Under the Hood
Two AI pipelines — extraction & search
📄
PDF Upload
Employee uploads resume → pdf-parse extracts raw text on server
↓
🤖
LLM Extraction
Groq Llama 3.3 70B reads resume → returns structured JSON: skills (name, category, proficiency, years), projects, certifications, LinkedIn URL
↓
⚙️
Skill Inference
Rules engine infers related skills — if Next.js found → React + JavaScript auto-added. 13 inference rules covering major tech stacks
↓
✅
HR Review Gate
Profile sits PENDING until HR approves. Human-in-the-loop ensures data quality. HR name recorded on every approval.
↓
🔍
Semantic Search
HR types natural language query → LLM reads all employee profiles → returns ranked list with match score 0–100 and written reasoning per candidate
✨ Both pipelines use Groq Llama 3.3 70B — called sequentially to stay within rate limits
